# Machine_Learning_Algorithm_Wheel
**机器学习算法纯手工轮子，主要目的在于深入理解算法原理和参数，包含：**
### 1. 分类
  - KNN
  - Naive Bayes
  - Decision Tree
  - Logistic Regression
  - SVM
  - AdaBoost
### 2. 回归
  - Linear Regression/Ridge Regression/Lasso Regression
  - Regression Tree：回归树/模型树
### 3. 聚类
  - KMeans/BiKMeans
### 4. 关联分析
  - Apriori：频繁项集、关联规则
  - FP-Growth：频繁项集
### 5. 降维
  - PCA
  - SVD
